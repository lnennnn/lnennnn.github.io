两两交换链表中的节点

题目描述： 给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。
题目链接/文章讲解/视频讲解： https://programmercarl.com/0024.%E4%B8%A4%E4%B8%A4%E4%BA%A4%E6%8D%A2%E9%93%BE%E8%A1%A8%E4%B8%AD%E7%9A%84%E8%8A%82%E7%82%B9.html

关键：cur 指向当前需要交换的两个节点的前一个节点（如链表为0->1->2->3，现在要交换2和3，则 cur 指向1），用两个临时结点分别保存 cur.next.next 和 cur.next.next.next 。

删除链表倒数第n个结点

题目描述： 给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。
题目链接/文章讲解/视频讲解：https://programmercarl.com/0019.%E5%88%A0%E9%99%A4%E9%93%BE%E8%A1%A8%E7%9A%84%E5%80%92%E6%95%B0%E7%AC%ACN%E4%B8%AA%E8%8A%82%E7%82%B9.html

这里采用 快慢双指针法 来定位倒数第n+1个结点（为什么是第n+1而不是第n个？因为删除结点需要先找到该节点的前一个结点，也就是需要让慢指针指向倒数第n+1个结点）。快指针向前移动n+1个结点，随后慢指针和快指针同时移动直至快指针指向空（None），则此时慢指针指向的结点即为倒数第n个结点的前一个结点。

链表相交

题目描述： 给你两个单链表的头节点 headA 和 headB ，请你找出并返回两个单链表相交的起始节点。如果两个链表没有交点，返回 null 。
题目链接/文章讲解：https://programmercarl.com/%E9%9D%A2%E8%AF%95%E9%A2%9802.07.%E9%93%BE%E8%A1%A8%E7%9B%B8%E4%BA%A4.html

如果两个链表相交，则相交的部分最多为 max(lenA, lenB) - abs(lenA - lenB) 。首先遍历求得两个列表的长度，接下来同样使用 快慢双指针法 分别指向两个链表的 dummyhead ，长度较长的那个链表指针先向前移动 abs(lenA - lenB) 个结点，这样两个指针如果以同样速度移动可以同时到达链尾。如果遍历到的当前结点的 next 和 val 均相同，则认为存在链表相交。

环形链表

题目描述： 给定一个链表的头节点 head ，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。
如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。如果 pos 是 -1，则在该链表中没有环。注意：pos 不作为参数进行传递，仅仅是为了标识链表的实际情况。不允许修改链表。
题目链接/文章讲解/视频讲解：https://programmercarl.com/0142.%E7%8E%AF%E5%BD%A2%E9%93%BE%E8%A1%A8II.html

这个题目比较有意思。该题可以分为两个步骤，首先是判断是否有环，如果有的话，再找到环的入口位置。
这里也是使用 快慢双指针法 ，不过这里快指针不是提前走，而是与慢指针同时出发，但快指针的速度比慢指针快，快指针一轮向前移动两次，而慢指针仅移动一次。如果有环，则快慢指针一定会在环的某处相遇；相遇后，此时令一个指针指向 dummyhead ，另一个指针指向相遇的位置，两个指针以相同的速度向前移动，则再次相遇的位置即为环的入口位置。
