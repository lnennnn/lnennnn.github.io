设计链表

题目描述： 你可以选择使用单链表或者双链表，设计并实现自己的链表。
单链表中的节点应该具备两个属性：val 和 next 。val 是当前节点的值，next 是指向下一个节点的指针/引用。
如果是双向链表，则还需要属性 prev 以指示链表中的上一个节点。假设链表中的所有节点下标从 0 开始。
实现 MyLinkedList 类：

  • MyLinkedList() 初始化 MyLinkedList 对象。
  • int get(int index) 获取链表中下标为 index 的节点的值。如果下标无效，则返回 -1 。
  • void addAtHead(int val) 将一个值为 val 的节点插入到链表中第一个元素之前。在插入完成后，新节点会成为链表的第一个节点。
  • void addAtTail(int val) 将一个值为 val 的节点追加到链表中作为链表的最后一个元素。
  • void addAtIndex(int index, int val) 将一个值为 val 的节点插入到链表中下标为 index 的节点之前。如果 index 等于链表的长度，那么该节点会被追加到链表的末尾。如果 index 比长度更大，该节点将 不会插入 到链表中。
  • void deleteAtIndex(int index) 如果下标有效，则删除链表中下标为 index 的节点。

题目链接/文章讲解/视频讲解：https://programmercarl.com/0707.%E8%AE%BE%E8%AE%A1%E9%93%BE%E8%A1%A8.html

单链表

首先需要定义链表节点类，参考第一部分「链表定义」中对单链表节点的定义。

在实现MyLinkedList的时候，首先需要使用__init__方法初始化类的属性，这里只需要 dummyhead 和 size 两个属性，dummyhead 指向链表头，size 表示链表的长度。

（1）获取链表中下标为index的节点的值
查找只需要逐个遍历，故临时指针 cur 指向链表的第一个节点，也就是 dummyhead.next ，循环 index 次即可找到下标为 index 的节点的值。在遍历前要记得先检查 index 值是否合法。

（2）将值为val的节点插入到链表中第一个元素之前
首先新建一个链表节点 new ，将 new 的后继置为head也就是 dummyhead.next ，再将 dummyhead.next 指向new即可。记得将 size 也要加1！！

（3）将值为val的节点追加到链表中作为链表的最后一个元素
同样新建一个节点 new 。注意增删改操作都需要先找到其前驱节点，即最终是对 cur.next 进行操作，所以先将 cur 初始化为 dummyhead ，即第一个节点的前驱节点。尾节点的特点是，其下标为 self.size - 1 ，即循环次数应该为 size 次，同时其后继节点为 None 。最后同样要注意更新 size 的值。

（4）将一个值为val的节点插入到链表中下标为index的节点之前
与插入到链尾类似，这里只需要改变循环终止条件，循环次数为 index 次，即可找到下标为 index 的节点，注意这里要先更新新节点的后继指针，再更新下标为 index 的节点的后继指针。注意操作前要先判断 index 的合法性，最后要更新 size 的值。

（5）删除链表中下标为index的节点
要删除一个节点，首先需要判断index的合法性，然后找到其前驱节点 pre ，将 pre 的后继指针指向起后继的后继即可。最后更新size的值。

反转链表

题目描述： 给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。
题目链接/文章讲解/视频讲解：https://programmercarl.com/0206.%E7%BF%BB%E8%BD%AC%E9%93%BE%E8%A1%A8.html

（1）双指针法

双指针法的双指针是指一个当前指针 cur 和当前指针的前驱指针 pre ，在对每一个指针进行反转的时候，需要先用一个临时指针 temp 保存 cur 在原链表中的后继节点，然后将 cur 和 pre 之间的指针方向进行反转，最后将 cur 赋值为 temp 对下一个节点进行处理。当 cur 为 None 时，表示链表已遍历完毕，循环结束。

（2）递归法

递归法是在双指针法的基础上，通过编写一个递归函数 reverse 来实现。reverse 传入的参数即为双指针法中的 pre 和 cur ，当cur 为 None 时直接返回 pre。
