最长公共子序列

dp数组的定义和 最长递增子数组 相同，递推关系发生改变：

  • text1[i-1] == text1[j-1] -> dp[i][j] = dp[i-1][j-1] + 1
  • text1[i-1] != text1[j-1] -> dp[i][j] = max(dp[i][j-1], dp[i-1][j])
    • 当末尾元素不同时，需要 状态延续 ，从相邻状态延续得到
    • dp[i][j-1]：不考虑text2[j-1]
    • dp[i-1][j]：不考虑text1[i-1]

其实和 最长公共子序列 是一样的，「不相交」即「有序」

判断子序列

可以转化为求最长公共子序列的长素，如果最长公共子序列的长度与 s 的长度相同，则说明 s 是为 t 的子序列

不同的子序列

求 s 中有多少个子序列 t，即求s变成t有多少种删除方式。

dp数组定义：

  • dp[i][j] 表示 s[:i-1] 中有多少个子序列 t[:j-1]

递推公式：

  • s[i-1] == t[j-1] -> dp[i-1][j-1] + dp[i-1][j]
    • dp[i-1][j-1]：让 s[i-1] 和 t[j-1] 匹配，这样问题就转化为 s[0:i-1] 和 t[0:j-1] 的子序列数（dp[i-1][j-1]）
    • dp[i-1][j]：不使用 s[i-1] 来匹配 t[j-1]，这样问题就转化为 s[0:i-1] 和 t[0:j] 的子序列数（dp[i-1][j]）。
  • s[i-1] != t[j-1] -> dp[i-1][j]
    • s[i-1] 无法与 t[j-1] 匹配，因此我们只能继承 s[0:i-1] 和 t[0:j] 的子序列数（dp[i-1][j]）。
    • 注意！这里只能是 i-1，模拟对s的删除操作，不能搞错顺序了！

dp数组初始化：

  • dp[i][0] = 1：dp[i][0] 表示 s 不为空 t 为空的情况，只有一种满足情况即把 s 中的字符全部删除，故方法数为1
  • dp[0][k] = 0：dp[0][j] 表示 s 为空 t 不为空的情况，无论怎样也无法满足，故方法数为0

两个字符串的删除操作

dp数组的定义：

  • dp[i][j] 表示使得字符串 word1[:i-1] 和 字符串 word2[:j-1] 相同的最小步数。

递推公式：

  • word1[i-1] == word2[j-1] -> 最后一个字符相同，则不需要进行删除操作，dp[i][j] 直接继承 dp[i-1][j-1]
  • word1[i-1] == word2[j-1] -> 需要进行删除操作
    • 删除 word1[i-1] ：dp[i-1][j] + 1
    • 删除 word2[j-1] ：dp[i][j-1] + 1
    • 都删除 ：dp[i-1][j-1] + 2

初始化：

  • dp[i][0] 表示第二个字符串为空，要想相等即将第一个字符串的元素全部删除；dp[0][j] 同理

编辑距离

和 两个字符串的删除操作 题不同的是，本题允许对字符进行添加和修改

  • 添加：与删除1个字符的效果其实是一样的，不用管
  • 修改：即在i-1和j-1相同的情况下，修改一个字符
    • dp[i][j] = dp[i-1][j-1] + 1
